# 💬 Chat App Backend

A real-time chat application backend built using Node.js, Express, MongoDB, and Socket.IO.

---

## 🚀 Features

- User Authentication (JWT)
- Real-time messaging (Socket.IO)
- Image upload (Cloudinary)
- MongoDB database
- Protected routes
- Online users tracking

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.IO
- JWT Authentication
- Cloudinary
